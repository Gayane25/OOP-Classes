### Task1

Create an `Author class` and a `Book class`.
`Author` should have:

- fields
- `name`
- `email`
- `gender`
- methods
- `getters` for fields
- ````javascript
  toString()```
  ````

`Book` should have:

- fields:
- `title`
- `author(Author)`
- `price`
- `quantity`
- methods:
- `getters` for fields
- `setters` for fields
- `getProfit() `- which calculates the profit from the book based on the `price` and `quantity`.
- `toString()`
